# Cookbook

Recetas **cortas y autocontenidas** para escenarios que aparecen una y otra vez en los GDDs. Cada una resuelve un problema específico — la lees en 2 minutos, copias el patrón y lo adaptas.

A diferencia de la [Referencia](/docs/referencia) (que describe **qué** hace cada feature), el Cookbook se enfoca en **cómo** combinar features para lograr un objetivo.
