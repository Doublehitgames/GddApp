# economyLink

El addon que conecta un ítem o contenido del juego con las monedas y variables globales que manejan su economía. Define precios de compra, venta, producción y desbloqueo en **una estructura unificada**.

## Por qué importa

Ítem barato, ítem caro, ítem premium, ítem desbloqueable por evento — todo eso es "economía". Sin un addon dedicado, dispersas esos números en campos sueltos por toda la página del ítem, y cuando necesitas rebalancear el juego (subir todos los precios un 10%, dar un 50% de descuento durante un evento), tienes que buscar en 80 lugares diferentes.

Este addon centraliza todo, y también referencia [`globalVariable`](/docs/referencia/addons/global-variable) para que puedas ajustarlo todo de una vez (ej.: una variable `item_buy_discount = 0.1` aplicada a todas las compras).

## Dónde suele vivir

- Páginas de tipo `🎒 Ítems` o `⚔️ Ítems con Efecto` — viene por defecto junto con Inventario.
- Páginas en blanco que representan algo vendible (una skin, un capítulo, un slot extra).

Singleton — 1 economyLink por página.

### `EconomyModifierRef`

```ts
{ refId: string }  // sectionId de una página globalVariable
```

Apunta a una `globalVariable` (ej.: `item_buy_discount`, `event_sell_markup`). El motor lee el valor de la variable en tiempo de ejecución y lo aplica sobre `buyValue`/`sellValue`.

## Ejemplo

Espada de Hierro — comprable y vendible, con descuento de temporada:

```json
{
  "id": "econ-iron-sword",
  "name": "Economía",
  "hasBuyConfig": true,
  "buyCurrencyRef": "section-gold",
  "buyValue": 100,
  "minBuyValue": 50,
  "buyModifiers": [
    { "refId": "section-buy-discount-var" }
  ],
  "hasSellConfig": true,
  "sellCurrencyRef": "section-gold",
  "sellValue": 50,
  "maxSellValue": 80,
  "sellModifiers": [
    { "refId": "section-sell-markup-var" }
  ],
  "hasUnlockConfig": false
}
```

## Bloques opcionales por defecto

Cada bloque (Compra, Venta, Producción, Desbloqueo) tiene un booleano `has*Config` que lo activa/desactiva. Los ítems de solo venta (no comprables) o solo desbloqueo (sin economía) resultan en un JSON exportado más limpio.

<Callout type="info" title="Modificadores globales creados automáticamente">
  Cuando creas una página de Ítem mediante wizard, automáticamente crea
  dos globalVariables (<code>item_buy_discount</code> e <code>item_sell_markup</code>)
  y referencia ambas en el economyLink. Esa es la forma predeterminada de la app de
  darte palancas de balance global sin que tengas que pensarlo.
</Callout>

## Wizard

No tiene wizard propio. Lo inicializa:

- `🎒 Ítems` (junto con Inventario)
- `⚔️ Ítems con Efecto` (junto con Inventario + AttributeModifiers)

El wizard pide una moneda en el diálogo de requisitos, y esa moneda se convierte automáticamente en `buyCurrencyRef` + `sellCurrencyRef`.
