# attributeDefinitions

Define los **atributos** del juego en una única lista que se convierte en la fuente de verdad para todas las páginas que trabajan con estadísticas.

## Por qué importa

Sin este addon, cada página de personaje inventa sus propias estadísticas — y seis meses después tienes 4 versiones de "ataque" (`atk`, `attack`, `dmg`, `Attack`). Aquí defines **una sola vez** que existen `hp`, `atk` y `def`, y todo en el proyecto referencia esas claves.

## Dónde suele vivir

- Una página de tipo `🎯 Atributos`. Es la configuración más común.
- Las páginas en blanco (creadas manualmente) también aceptan el addon.

Normalmente tienes **solo una** página de atributos en todo el proyecto. También son posibles múltiples perfiles (combate, social, magia) si necesitas mantener conjuntos de estadísticas separados.

## Ejemplo

Un perfil de RPG clásico:

```json
{
  "id": "attr-defs-1",
  "name": "Atributos Base",
  "attributes": [
    { "id": "1", "key": "hp",  "label": "HP",  "valueType": "int", "defaultValue": 100, "min": 0 },
    { "id": "2", "key": "atk", "label": "ATK", "valueType": "int", "defaultValue": 10,  "min": 0 },
    { "id": "3", "key": "def", "label": "DEF", "valueType": "int", "defaultValue": 5,   "min": 0 },
    { "id": "4", "key": "spd", "label": "SPD", "valueType": "int", "defaultValue": 5,   "min": 0 }
  ]
}
```

## Quién apunta a este addon

Casi todos los addons "conscientes de atributos" referencian un `attributeDefinitions` mediante el campo `definitionsRef`:

- `attributeProfile.definitionsRef` — define qué claves puede completar el perfil
- `attributeModifiers.definitionsRef` — define qué claves puede modificar el modificador
- `progressionTable` (cuando se usa para escalar atributos) — referencia indirectamente
- `skills.SkillCost.definitionsRef` (cuando el tipo de costo es `attribute`) — qué perfil define la clave

<Callout type="tip" title="Múltiples perfiles">
  Si tu juego tiene grupos de estadísticas independientes (ej.: combate vs social),
  crea **múltiples** páginas de Atributos. Cada modificador/habilidad apunta
  a exactamente uno — no hay conflicto.
</Callout>

## Wizard

El tipo de página `🎯 Atributos` crea este addon **pre-poblado** con 4 atributos comunes (HP, ATK, DEF, SPD). Puedes aceptarlos, eliminarlos o agregar más en el momento de creación mediante el selector de slots. Opciones de personalización:

- **Slots por defecto**: casillas para activar/desactivar HP/ATK/DEF/SPD
- **Slots personalizados**: hasta N campos extra con `key` + `label` + `valueType`

Todo esto se convierte en el array `attributes[]` de la entrada final.
