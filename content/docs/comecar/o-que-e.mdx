# O que é o GDD Manager

<Callout type="warning" title="Em breve">
  Esta página é um stub — o conteúdo completo chega na próxima rodada.
</Callout>

Um app pra escrever Game Design Documents que ficam **vivos** e **conectados** ao seu jogo.

## Por que importa

Documentos de design tradicionais (Word, Notion, PDF) ficam desatualizados rápido porque vivem **separados** do código. O GDD Manager tenta fechar essa distância: você descreve o sistema **uma vez**, e o app exporta os dados que o motor consome.
