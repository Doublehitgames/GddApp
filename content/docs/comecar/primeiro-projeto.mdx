# Seu primeiro projeto

<Callout type="warning" title="Em breve">
  Esta página é um stub — o walkthrough completo chega na próxima rodada com
  screenshots de cada passo.
</Callout>

Vamos criar um GDD pequeno (10-15 páginas) pra você sentir o fluxo. O exemplo é um RPG de fantasia com economia simples.

## Pré-requisitos

- Conta no GDD Manager (cadastro grátis)
- 15 minutos

## Passos previstos no walkthrough

1. Criar um projeto vazio
2. Página de **Atributos** (HP, ATK, DEF) com o wizard
3. Página de **Moeda** (Ouro)
4. Página de **Habilidade** vinculando aos atributos
5. Página de **Item** com efeito (Espada de Ferro)
6. Configurar **Remote Config** pra exportar tudo
