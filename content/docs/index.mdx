## O que existe no app

O GDD Manager se organiza em três camadas:

1. **Páginas** — cada uma representa um conceito do seu jogo (uma classe, uma habilidade, uma economia). Tipadas (com wizard) ou em branco.
2. **Addons** — blocos especializados dentro da página. Skills, Currency, Modificadores de Atributos, Tabela de Progressão, etc. Hoje são 16+ tipos.
3. **Remote Config** — exporta tudo pra JSON consumível pelo seu motor (Unity, Godot, Unreal, web). Schema flexível, formatos múltiplos.

A força do app está na **integração entre as camadas**: uma skill referencia um modificador de atributos, que aponta pra uma definição de atributos, que vira um campo num JSON exportado — tudo navegável e validado.
