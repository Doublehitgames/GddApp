# Addons

Um **addon** é um bloco semântico dentro de uma página: tem schema próprio, painel de edição e contribui pro JSON exportado. A maioria é singleton (1 por página). As exceções são `richDoc` e `exportSchema`.

Cada addon abaixo tem sua própria página de referência. Os links levam pra elas.

<Callout type="info" title="Singleton vs múltiplos">
  Quase todos os addons só permitem **1 instância por página**. Isso garante
  que refs internas (Skills → Modifiers da mesma página) sempre tenham um
  alvo único. As exceções são `richDoc` (você pode quebrar conteúdo em vários
  blocos) e `exportSchema` (uma página pode produzir múltiplos arquivos JSON).
</Callout>
